# RestaurantFlow

## Slogan

Simplificando a operação do seu restaurante.

## Objetivo

RestaurantFlow é uma plataforma web desenvolvida para otimizar a gestão operacional de restaurantes, oferecendo uma experiência moderna, intuitiva e eficiente para gerentes e equipes.

## Público-alvo

- Restaurantes
- Hamburguerias
- Cafeterias
- Pizzarias
- Bistrôs

## Valores

- Simplicidade
- Agilidade
- Organização
- Eficiência
- Experiência do usuário
